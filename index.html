Propósito

.

El propósito de este portafolio de evidencias es tener un lugar donde poder guardar todas las actividades. Realizadas en el curso de "Seguridad Informática"; además, este portafolio tendrá una función adicional. Ya que en él podré mostrar el avance de los conocimientos adquiridos a lo largo del curso, así como también. Tener un registro de las actividades que fueron entregadas, así mismo, para el final del curso este. El portafolio me servirá para tener una prueba clara de todo lo que se vio en el curso, esto para asegurar. Que todos los conocimientos que día con día adquirí los puse en práctica para mejorar mis habilidades. En la rama de seguridad informática.

Enfoque

El enfoque del curso es aprender sobre los temas de seguridad informatica, pruebas de penetración y analisis de vulnerabilidades, con todo esto dicho se veran temas que ayuden a apmpliar el conocimiento en estas areas, asi como el manejo de la seguridad informatica, para poder tener una mejor seguridad en entornos reales, esto tambien para tener un buen sistemma de prevencion ante problemas de este ámbito que puedan surgir.

Importancia

La importancia del portafolio es el poder guardar todas las evidencias de las actividades realizadas en el curso, asi como tambien el tener un registro de todo el conocimiento adquirido a lo largo del mismo, este portafolio me sirve para tener un espacio donde poder tener todo lo que trabaje y al final poder mostrarlo.
